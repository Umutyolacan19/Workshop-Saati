☕ Workshop Saati Akademi - Eğitim ve Rezervasyon Platformu

Bu proje, Sistem Analizi ve Tasarımı dersi kapsamında geliştirilmiş; kahve akademileri, eğitmenler ve öğrenciler için tasarlanmış uçtan uca bir Eğitim (LMS) ve Atölye Rezervasyon Yönetim Sistemidir.

🎯 Projenin Amacı

Geleneksel barista atölyeleri ve eğitim kayıt süreçlerini dijitalleştirmek. Ziyaretçilere kurumsal bir web sitesi sunarken, arka planda dinamik bir öğrenci/eğitmen yönetim portalı (Dashboard) sağlamak.

🚀 Öne Çıkan Özellikler

Çoklu Dil Desteği (TR/EN): Sitenin arayüzü anlık olarak Türkçe ve İngilizce arasında değiştirilebilir. (Google Çeviri çakışmalarına karşı translate="no" koruması eklenmiştir.)

Rol Bazlı Erişim (Ziyaretçi, Öğrenci, Yönetici): Her kullanıcı tipine özel, izole edilmiş paneller ve menüler.

Dinamik Rezervasyon ve Kontenjan Algoritması: Kontenjanı dolan eğitimlere yeni kayıtların otomatik engellenmesi.

Sertifika Üretimi: Tamamlanan eğitimler için öğrencilere özel, doğrudan PDF olarak yazdırılabilir çerçeveli başarı sertifikası.

Geri Bildirim Sistemi: Eğitimi tamamlayan öğrencilerin 5 yıldız üzerinden değerlendirme ve yorum yapabilmesi.

Güvenlik Kalkanı (Error Boundary): Beklenmedik çökmeleri engelleyen ve kullanıcıya özel hata ekranı sunan React koruma katmanı.

Çevrimdışı (Local) Destek: Veritabanı (Firebase) bağlantısı olmasa dahi tüm sistemin RAM üzerinde kusursuz simüle edilebilmesi.

🛠️ Kullanılan Teknolojiler

Frontend: React.js (Vite ile derlenmiştir)

Stil: Tailwind CSS (v4)

İkonlar: Lucide React & Custom SVG Icons

Veritabanı (Opsiyonel): Firebase Firestore

💻 Kurulum ve Çalıştırma (Hocalar İçin)

React projelerinde .exe dosyası bulunmaz. Projeyi çalıştırmak için:

Bilgisayarınızda Node.js kurulu olmalıdır.

Terminali bu klasör dizininde açın.

Bağımlılıkları yüklemek için: npm install

Projeyi başlatmak için: npm run dev

Terminalde verilen yerel adrese (örn: http://localhost:5173) tıklayarak siteye erişebilirsiniz.
